需要改动的文件包括index.html和main.js

index.html

	1. 自动完成控件，在容器添加select 类即可实现列表显示
	2. 自定义选择控件，在容器添加select 类可以显示列表，自定义选择控件需添加ctms类

main.js

	1. resources数组为需要加载的资源

	2. buildingsCoordinates 为可点击区域多边形

	3. mouseoverEvents 为对应的建筑，包括的选择器和点击事件处理

	4. 注意：mouseoverEvents和buildingsCoordinates的顺序要对应！
